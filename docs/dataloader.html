Data Loader¶

contk.dataloader provides classes and functions downloading and loading benchmark data automatically. It reduce your cost preprocessing data and provide a fair dataset for every model. It also help you adapt your model from one dataset to other datasets.

class contk.dataloader.SingleTurnDialog(_key_name=None)[source]¶

Base class for single-turn dialog datasets.

ext_vocab¶

special tokens, must be placed at beginning of vocab_list. default: [“<pad>”, “<unk>”, “<go>”, “<eos>”]

Type:list
pad_id¶

token for padding. default: 0

Type:int
unk_id¶

token for unkown words. default: 1

Type:int
go_id¶

token at the beginning of sentences. default: 2

Type:int
eos_id¶

token at the end of sentences. default: 3

Type:int
key_name¶

name of subsets of the data. default: [“train”, “dev”, “test”]

Type:list
vocab_list¶

all tokens of the datasets.

Type:list
word2id¶

a dict mapping tokens to index. Maybe you want to use sen_to_index() instead.

Type:dict

Note

(For developer) You must initialize following attributes in the subclasses.

  • vocab (list): don’t forget ext_vocab.
  • word2id (dict): should be initialized by {w: i for i, w in enumerate(self.vocab_list)}
  • data (dict): a dict mapping key_name to lists, which contains sentences in index form.
  • index (dict): a dict mapping key_name to lists, should be initialized by list(range(len(data[key])))

class contk.dataloader.OpenSubtitles(file_path, min_vocab_times=10, max_sen_length=50)[source]¶

A dataloder for OpenSubtitles dataset.

Parameters:
  • file_path (str) – a str indicates the dir of OpenSubtitles dataset.
  • min_vocab_times (int) – A cut-off threshold of UNK tokens. All tokens appear less than min_vocab_times will be replaced by <unk>. Default: 10.
  • max_sen_length (int) – All sentences longer than max_sen_length will be shortened to first max_sen_length tokens. Default: 50.

Refer to SingleTurnDialog for other attributes.
