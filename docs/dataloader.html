Data Loader¶

contk.dataloader provides classes and functions downloading and loading benchmark data automatically. It reduce your cost preprocessing data and provide a fair dataset for every model. It also help you adapt your model from one dataset to other datasets.

Base class for single-turn dialog datasets. This is an abstract class.

Parameters:

get_batch(key, index)[source]¶

Get a batch of specified index.

Arguments:
key (str): must be contained in key_name index (list): a list of specified index
Returns:
A dict at least contains post, post_length, resp, resp_length. See the example belows.

Examples

vocab_list = [“<pad>”, “<unk>”, “<go>”, “<eos>”, “how”, “are”, “you”, “hello”, “i”, “am”, “fine”]
data = { “train”:{ “post”: [ [2, 4, 5, 6, 3], # first post: <go> how are you <eos> [2, 7, 3], # second post: <go> hello <eos> ], “resp”: [ [2, 8, 9, 10, 3], # first response: <go> i am fine <eos> [2, 7, 3], # second response: <go> hello <eos> ], }, “dev”: {…}, # similar to train “test”: {…}, # similar to train } >>> dataloader.get_batch(‘train’, [0, 1]) >>> { “post”: [ [2, 4, 5, 6, 3], # first post [2, 7, 3, 0, 0], # second post with <pad> ], “resp”: [ [2, 8, 9, 10, 3], # first response [2, 7, 3, 0, 0], # second response with <pad> ], “post_length”: [5, 3], # length of posts “resp_length”: [5, 3], # length of responses }

Examples

trim_index(index)[source]¶
Trim index. There will be two steps:
  • If there is an <eos> in sentences, find first <eos> and abondon words after it (included the <eos>).
  • ignore <pad> s at the end of the sentence.
Parameters:index (list) – a list of int

Examples

vocab_list = [“<pad>”, “<unk>”, “<go>”, “<eos>”, “I”, “have”, “been”, “to”, “Sichuan”] >>> dataloader.trim_index( … [2, 4, 5, 6, 7, 8, 0, 0, 3, 4, 3, 0]) # <go> I have been to Sichuan <pad> <pad> <eos> I <eos> <pad> >>> [2, 4, 5, 6, 7, 8] # <go> I have been to Sichuan

class contk.dataloader.OpenSubtitles(file_path, min_vocab_times=10, max_sen_length=50)[source]¶

A dataloder for OpenSubtitles dataset.

Parameters:
  • file_path (str) – a str indicates the dir of OpenSubtitles dataset.
  • min_vocab_times (int) – A cut-off threshold of UNK tokens. All tokens appear less than min_vocab_times will be replaced by <unk>. Default: 10.
  • max_sen_length (int) – All sentences longer than max_sen_length will be shortened to first max_sen_length tokens. Default: 50.

Refer to SingleTurnDialog for attributes.

Reference:
[1] http://opus.nlpl.eu/OpenSubtitles.php [2] P. Lison and J. Tiedemann, OpenSubtitles2016: Extracting Large Parallel Corpora from Movie and TV Subtitles.(LREC 2016)
